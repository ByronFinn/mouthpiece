嘴替 (Mouthpiece) 隐私政策

最后更新 / Last updated: 2025-05-17

中文 English

概述

嘴替是一款 Chrome 浏览器扩展，帮助用户为选中的网页文本生成多风格社交媒体评论。我们重视您的隐私。

我们收集什么数据

  • 账号信息：注册时的用户名和密码。密码以加盐 SHA-256 哈希存储，我们无法查看您的明文密码。
  • API Key：系统自动生成，用于标识您的身份。存储在本地浏览器和服务器数据库中。

我们如何使用数据

  • 用户名和密码仅用于身份验证。
  • 您选中的文本和图片会发送到我们的服务器，再转发至第三方 LLM API 生成评论。这些数据不会被存储。
  • 生成的评论在返回给您后，不会在我们的服务器上保留。

我们不做什么

  • 不收集浏览历史
  • 不收集cookies
  • 不追踪用户行为
  • 不出售、共享或转让用户数据
  • 不在服务器上存储您选中的文本、图片或生成的评论

数据存储

用户账号信息存储在我们自管的服务器 SQLite 数据库中。您的浏览器本地仅存储 API Key（通过 Chrome Storage API）。

第三方服务

我们使用第三方 LLM API 处理评论生成请求。发送的内容受该服务提供商的隐私政策约束。

用户权利

  • 您可以随时通过插件重置 API Key
  • 您可以联系管理员删除您的账号和所有关联数据

政策变更

如本政策发生变更，我们会在此页面更新，并在插件更新说明中注明。

如有疑问，请联系：mouthpiece@baifan.site

Mouthpiece Privacy Policy

Last updated: 2025-05-17

中文 English

Overview

Mouthpiece is a Chrome browser extension that generates multi-style social media comments for selected webpage text. We value your privacy.

What We Collect

  • Account Information: Username and password at registration. Passwords are stored as salted SHA-256 hashes — we cannot view your plaintext password.
  • API Key: Auto-generated by the system to identify you. Stored locally in your browser and on our server database.

How We Use Data

  • Username and password are used solely for authentication.
  • Selected text and images are sent to our server, then forwarded to a third-party LLM API for comment generation. This data is not stored.
  • Generated comments are not retained on our servers after being returned to you.

What We Don't Do

  • We do not collect browsing history
  • We do not collect cookies
  • We do not track user behavior
  • We do not sell, share, or transfer user data
  • We do not store your selected text, images, or generated comments on our servers

Data Storage

User account information is stored in a self-hosted SQLite database. Your browser stores only the API key locally via the Chrome Storage API.

Third-Party Services

We use third-party LLM APIs to process comment generation requests. Content sent to these services is subject to their respective privacy policies.

User Rights

  • You can reset your API Key at any time through the extension
  • You can contact the administrator to delete your account and all associated data

Policy Changes

If this policy changes, we will update this page and note the changes in the extension's release notes.

Questions? Contact: mouthpiece@baifan.site
