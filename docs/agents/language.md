# 文档语言

工程技能为本仓库撰写**面向人类的输出**时应使用的语言。

## 语言

**中文**

技能产出的所有正文——PRD（`docs/prd/`）、ADR（`docs/adr/`）、`CONTEXT.md`、Issue 标题与正文、评审报告与评论——均使用此语言撰写。

## 范围

- **适用**：文档与 Issue 中的面向人类的正文。
- **不适用**：代码、标识符、文件名、CLI 命令与配置值。这些遵循代码惯例，不受本设置约束。

## 当技能要求「撰写输出」时

使用上述语言撰写正文。不要默默切换到当前对话的语言——团队阅读的是配置的语言，可能与触发技能的成员不同。

若某份产物确实需要第二种语言（例如双语发布说明），产出技能可以偏离，但配置的语言始终是主要语言。

## 若本文件缺失

回退到用户的输入语言，并询问团队偏好哪种语言以便记录于此。不要在每个会话中反复猜测。
